# Ayush Trilokchandani - Personal Portfolio

Welcome to the repository for my personal portfolio website! This site showcases my skills, experience, and the software development projects I have built. 

You can view the live site here: **[https://ayusht20.github.io](https://ayusht20.github.io)**

## 🚀 Features

- **Modern & Responsive UI:** Fully responsive design built from scratch without a heavy CSS framework.
- **Dynamic Animations:** Features smooth scroll-reveal animations, staggered load effects, and interactive hover states.
- **Dark Theme:** A sleek, developer-focused dark mode aesthetic with indigo accents.
- **Project Showcase:** Highlights my best work, complete with live Vercel deployment links and GitHub repository links.

## 🛠️ Technologies Used

- **HTML5:** Semantic HTML structure.
- **CSS3:** Custom vanilla CSS utilizing CSS Variables, Flexbox, CSS Grid, and Keyframe Animations.
- **JavaScript (Vanilla):** DOM manipulation, event handling, and Intersection Observer API for scroll animations.

## 💻 Running Locally

To run this project locally, simply clone the repository and open `index.html` in your browser.

```bash
# Clone the repository
git clone https://github.com/Ayusht20/Ayusht20.github.io.git

# Navigate into the directory
cd Ayusht20.github.io

# Open index.html in your preferred browser, or use a live server extension.
```
